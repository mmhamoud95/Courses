```
🎯

Faithfulness (fidélité)

La réponse est-elle fondée uniquement sur les documents récupérés ? Score 0–1.

🔍

Context Recall

Le retriever a-t-il récupéré tous les chunks nécessaires pour répondre ? Score 0–1.

💬

Answer Relevancy

La réponse répond-elle vraiment à la question ? Score 0–1.

✅

Factual Correctness

Les faits dans la réponse sont-ils corrects par rapport à la référence ? Score 0–1.

MétriqueQuestion poséeBesoin référence ?Seuil recommandé
FaithfulnessLe LLM invente-t-il ?Non ✓> 0.80
Context RecallLe retriever est-il complet ?Oui (référence)> 0.75
Answer RelevancyHors sujet ?Non ✓> 0.80
Factual CorrectnessLes faits sont-ils justes ?Oui (référence)> 0.70
```

```
⚠️
Piège fondamental — Le keyword match ne suffit pas

Un score de 96% en keyword match peut cacher des hallucinations graves. Un LLM peut reprendre les bons mots-clés tout en inventant des chiffres ou des règles. C'est précisément pour ça que RAGAS et LLM-as-Judge existent — ils évaluent le sens, pas les mots.

🎓 Pour enseigner l'évaluation

Commence par montrer un exemple de "bonne réponse en apparence" qui est en fait une hallucination. Ex : la réponse dit "3 jours de télétravail" alors que la doc dit "2 jours". Le keyword score reste élevé car "jours" et "télétravail" sont présents. Ça crée l'urgence d'apprendre des métriques sémantiques.
```

# ── INITIALISER LES MÉTRIQUES ─────────────────────────────────────────
metrics = [
Faithfulness(llm=evaluator_llm),            # hallucinations ?
LLMContextRecall(llm=evaluator_llm),         # retriever complet ?
AnswerRelevancy(llm=evaluator_llm, embeddings=evaluator_emb),  # hors sujet ?
FactualCorrectness(llm=evaluator_llm),       # faits corrects ?
]

# ── CHARGER LE DATASET DU J1 ──────────────────────────────────────────

async def evaluer_sample(sample_dict: dict) -> dict:
“Évalue un sample sur toutes les métriques”
sample = SingleTurnSample(
user_input=sample_dict[“user_input”],
retrieved_contexts=sample_dict[“retrieved_contexts”],
response=sample_dict[“response”],
reference=sample_dict.get(“reference”),

for metric in metrics:

scores[metric.name] = round(score, 3)
except Exception as e:

return scores

```
✅ Quand utiliser LLM-as-Judge
  • RAGAS = pour le RAG (retriever + génération). LLM-as-Judge = pour tout le reste (agents, chatbots, MCP...)
  • AgentGoalAccuracy : l'agent a-t-il atteint l'objectif de l'utilisateur ?
  • DiscreteMetric de RAGAS : créer une métrique custom avec un prompt
  • Les biais LLM-as-Judge : le modèle préfère ses propres réponses, les réponses longues, les premières options
⚠️ Biais à connaître
  • Self-enhancement bias : Mistral note mieux les réponses générées par Mistral
  • Length bias : les réponses longues sont mieux notées même si moins précises
  • Position bias : la première option est souvent préférée dans les comparaisons
  • Solution : utiliser un LLM évaluateur différent du LLM générant les réponses
```

# ── MÉTHODE 1 : Génération via prompt simple (plus rapide) ─────────────
async def generer_questions_simple(document_text: str, nb_questions: int = 10) -> list:
””“Génère des questions de test depuis un document via prompt Mistral.”””
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
import re

```
llm = ChatOllama(model="mistral", temperature=0.4)
prompt = ChatPromptTemplate.from_messages([
    ("system", f"""Génère exactement {nb_questions} paires question/réponse en français basées sur ce document.
```

Format STRICT (une paire par ligne) :
Q: [question factuelle]
R: [réponse courte et précise]

Document : {document_text[:2000]}”””),
(“human”, “Génère les questions maintenant.”),
])
raw = await (prompt | llm | StrOutputParser()).ainvoke({})

```
# Parser les paires Q/R
pairs = []
lines = raw.strip().split("\n")
q, r = None, None
for line in lines:
    if line.startswith("Q:"): q = line[2:].strip()
    elif line.startswith("R:") and q:
        r = line[2:].strip()
        pairs.append({"question": q, "reference": r})
        q, r = None, None
return pairs[:nb_questions]
```


import asyncio, json
from datetime import datetime
from pathlib import Path
from langchain_ollama import ChatOllama
from langchain_chroma import Chroma
from langchain_huggingface import HuggingFaceEmbeddings
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnablePassthrough
from ragas.llms import LangchainLLMWrapper
from ragas.embeddings import LangchainEmbeddingsWrapper
from ragas.dataset_schema import SingleTurnSample
from ragas.metrics import Faithfulness, LLMContextRecall, AnswerRelevancy

# ── SEUILS DE RÉGRESSION ─────────────────────────────────────────────
THRESHOLDS = {
“faithfulness”: 0.80,       # alerte si baisse sous 0.80
“context_recall”: 0.75,
“answer_relevancy”: 0.80,
}
REGRESSION_DELTA = 0.05  # alerte si baisse de plus de 5% par rapport au baseline

# ── RAG À ÉVALUER (version actuelle) ─────────────────────────────────
def build_rag():
emb = HuggingFaceEmbeddings(model_name=“all-MiniLM-L6-v2”)
vectordb = Chroma(persist_directory=”./base_entreprise”, embedding_function=emb)
llm = ChatOllama(model=“mistral”, temperature=0)
retriever = vectordb.as_retriever(search_kwargs={“k”: 3})

```
def format_docs(docs): return "\n\n".join(d.page_content for d in docs)
prompt = ChatPromptTemplate.from_messages([
    ("system", "Réponds UNIQUEMENT avec le contexte. Cite la source.\n\nContexte:\n{context}"),
    ("human", "{question}"),
])
chain = ({"context": retriever | format_docs, "question": RunnablePassthrough()} | prompt | llm | StrOutputParser())
return chain, retriever
```

async def run_pipeline(dataset_path: str, baseline_path: str = None):
dataset = json.loads(Path(dataset_path).read_text())
chain, retriever = build_rag()
eval_llm = LangchainLLMWrapper(ChatOllama(model=“mistral”, temperature=0))
eval_emb = LangchainEmbeddingsWrapper(HuggingFaceEmbeddings(model_name=“all-MiniLM-L6-v2”))
metrics = [Faithfulness(llm=eval_llm), LLMContextRecall(llm=eval_llm), AnswerRelevancy(llm=eval_llm, embeddings=eval_emb)]

```
results = []
print(f"🔍 Évaluation de {len(dataset)} samples...\n")
for i, s in enumerate(dataset, 1):
    print(f"  [{i}/{len(dataset)}] {s['user_input'][:60]}")
    # Générer la réponse avec le RAG actuel
    response = chain.invoke(s["user_input"])
    contexts = [d.page_content for d in retriever.invoke(s["user_input"])]
    sample = SingleTurnSample(user_input=s["user_input"], retrieved_contexts=contexts, response=response, reference=s.get("reference"))

    scores = {"question": s["user_input"], "response": response[:100]}
    for m in metrics:
        try: scores[m.name] = round(await m.single_turn_ascore(sample), 3)
        except: scores[m.name] = None
    results.append(scores)

# ── CALCUL DES MOYENNES ────────────────────────────────────────────
metric_names = [m.name for m in metrics]
averages = {}
for m in metric_names:
    vals = [r[m] for r in results if r.get(m) is not None]
    averages[m] = round(sum(vals)/len(vals), 3) if vals else 0

# ── DÉTECTION DE RÉGRESSION ───────────────────────────────────────
regressions = []
if baseline_path and Path(baseline_path).exists():
    baseline = json.loads(Path(baseline_path).read_text())
    baseline_avg = baseline.get("averages", {})
    for m, score in averages.items():
        threshold = THRESHOLDS.get(m, 0.7)
        baseline_score = baseline_avg.get(m, score)
        if score < threshold:
            regressions.append(f"⚠️ {m} = {score:.3f} (seuil: {threshold})")
        elif baseline_score - score > REGRESSION_DELTA:
            regressions.append(f"📉 {m} : {baseline_score:.3f} → {score:.3f} (régression !)")

# ── RAPPORT FINAL ─────────────────────────────────────────────────
run_id = datetime.now().strftime("%Y%m%d_%H%M")
report = {"run_id": run_id, "nb_samples": len(dataset), "averages": averages, "regressions": regressions, "details": results}
Path("eval/results").mkdir(exist_ok=True)
report_path = f"eval/results/run_{run_id}.json"
Path(report_path).write_text(json.dumps(report, ensure_ascii=False, indent=2))

print(f"\n{'='*50}\n📊 RAPPORT — Run {run_id}")
for m, v in averages.items():
    status = "✅" if v >= THRESHOLDS.get(m, 0.7) else "❌"
    print(f"  {status} {m:<25} {v:.3f}")
if regressions:
    print("\n🚨 RÉGRESSIONS DÉTECTÉES :")
    for r in regressions: print(f"  {r}")
else:
    print("\n✅ Aucune régression détectée")
print(f"\n✓ Rapport sauvegardé : {report_path}")
return report
```

import streamlit as st, json, glob

# ── CHARGER TOUS LES RAPPORTS ─────────────────────────────────────────

if metric_cols:
st.line_chart(df.set_index(“run_id”)[metric_cols])

# ── ALERTES RÉGRESSION ────────────────────────────────────────────────
latest = runs[-1] if runs else {}
